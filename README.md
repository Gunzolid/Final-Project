# English Documentation

## Overview
Park Nhai helps users locate available parking spaces in real-time, reducing the time and stress associated with finding parking. The system integrates with IoT sensors to provide accurate, up-to-the-minute information on parking availability. Users can also manually reserve spots directly from the map.

## Key Features

### 1. Real-time Map & Availability
- View the parking lot layout with real-time status updates.
- **Color Indicators:**
  - üü¢ **Green (Available):** The spot is free. You can tap to hold it.
  - üî¥ **Red (Occupied):** The spot is currently occupied by a car.
  - üü† **Orange (Held):** The spot is temporarily reserved by a user (15-minute hold).
  - üîò **Grey (Unavailable):** The spot is offline or disabled by admins.

### 2. Manual Spot Selection
- Users can zoom and pan interactively on the parking map.
- **Tap to Hold:** Simply tap on any **Green** spot to reserve it for 15 minutes.
- The system will guide you to log in if necessary.

### 3. Navigation
- Integrated with Google Maps to provide turn-by-turn navigation to your selected parking spot.

### 4. Admin Dashboard
- **Live Monitoring:** Admins can view the status of all parking spots.
- **Statistics:** View charts for daily usage and revenue (if applicable).
- **Spot Management:** Admins can manually disable spots (e.g., for maintenance).

## Tech Stack
- **Frontend:** Flutter
- **Backend:** Firebase (Auth, Firestore, Cloud Functions)
- **IoT:** Smart sensors for vehicle detection

## Getting Started

### Prerequisites
- Flutter SDK
- Dart SDK

### Installation
1. Clone this repository.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the application.
