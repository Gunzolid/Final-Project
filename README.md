# Park Nhai (ปาร์คไหน)

**Park Nhai** is a Real-time parking search application.
**Park Nhai** คือแอปพลิเคชันค้นหาที่จอดรถแบบเรียลไทม์

---

## Overview / ภาพรวม

Park Nhai helps users locate available parking spaces in real-time, reducing the time and stress associated with finding parking. The system integrates with IoT sensors to provide accurate, up-to-the-minute information on parking availability.

Park Nhai ช่วยให้ผู้ใช้ค้นหาที่จอดรถว่างได้ในทันที ช่วยลดเวลาและความเครียดในการวนหาที่จอดรถ ระบบทำงานร่วมกับเซ็นเซอร์ IoT เพื่อให้ข้อมูลสถานะที่จอดรถที่แม่นยำและเป็นปัจจุบันที่สุด

---

## Features / ฟีเจอร์หลัก

- **Real-time Availability:** View available parking spots instantly.
  - **ตรวจสอบสถานะเรียลไทม์:** ดูจุดจอดรถที่ว่างได้ทันทีผ่านแอป
- **Map:** View parking locations on an interactive map.
  - **แผนที่:** แสดงตำแหน่งลานจอดรถบนแผนที่
- **Navigation:** Get directions to your chosen parking spot.
  - **ระบบนำทาง:** นำทางไปยังจุดจอดรถที่เลือก
- **Admin Dashboard:** Manage parking lots and view analytics.
  - **แดชบอร์ดผู้ดูแลระบบ:** สำหรับจัดการลานจอดรถและดูสถิติการใช้งาน

---

## Tech Stack / เทคโนโลยีที่ใช้

- **Frontend:** Flutter
- **Backend:** Firebase
- **IoT:** Smart sensors for vehicle detection (Smart sensors สำหรับตรวจจับรถยนต์)

## Getting Started / การเริ่มต้นใช้งาน

This project is built with Flutter.
โปรเจกต์นี้พัฒนาด้วย Flutter

### Prerequisites / สิ่งที่ต้องมี

- Flutter SDK
- Dart SDK

### Installation / การติดตั้ง

1. Clone this repository.
   - Clone repository นี้ลงมา
2. Run `flutter pub get` to install dependencies.
   - รันคำสั่ง `flutter pub get` เพื่อติดตั้ง dependencies
3. Run `flutter run` to launch the application.
   - รันคำสั่ง `flutter run` เพื่อเปิดแอปพลิเคชัน

For help getting started with Flutter development, view the [online documentation](https://docs.flutter.dev/).
สำหรับข้อมูลเพิ่มเติมเกี่ยวกับการพัฒนา Flutter สามารถดูได้ที่ [online documentation](https://docs.flutter.dev/)
